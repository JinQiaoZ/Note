# 在Windows下安装与使用Jekyll

简单的介绍在Windows系统下如何安装Ruby和Jekyll，以及怎么样使用Jekyll简单的进行创建和运行Blog项目

# 安装

# 安装Ruby

下载网站：https://rubyinstaller.org/downloads/ (opens new window)，这里要注意，安装有配套DevKit的版本，如Ruby+Devkit 2.4.4-2(x64)
安装路径D:\Tools，过程简单的I accept，Next，Finish即可
打开CMD，输入如下命令查看Ruby版本，查看是否安装成功

ruby -v

显示

ruby 2.4.4p296 (2018-03-28 revision 63013) [x64-mingw32]
