# Java程序员的悲哀

我写了无数个面向对象的程序 最终还是找不到对象

我会Java这么美的编程语言 却没能打动你的芳心

我构造了许多方法 却没有办法让你爱上我

你不是我的Private 更不是我可以调用的Public 你只是他一个人的Protected

我Try着靠近你你却爱答不理

我想Catch你的心但也没有Return

Finally!我哭着对天空说:Hello World!
