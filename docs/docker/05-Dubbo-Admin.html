# Docker下Dubbo-Admin的使用

dubbo-admin 是官方提供的一个界面管理控制台，下载地址: Github

# 1. 前言

可以自行下载到本地打包启动，详细启动可以查询文档: https://github.com/apache/dubbo-admin/blob/develop/README_ZH.md，这里我是采用 Docker 的方式进行启动

# 2. 镜像

我们可以去 Docker Hub 查询版本号，然后直接启动 Docker，执行命令 docker pull apache/dubbo-admin:latest 下载最新版本镜像，使用命令 docker images 查看下载的镜像

# 3. 启动

查看: Docker Hub，docker-compose 直接启动

version: '3'
services:
  zk:
    image: zookeeper
    container_name: zk
    ports:
      - 2181:2181
  dubbo-admin:
    image: apache/dubbo-admin
    container_name: dubbo-admin
    # 等待zk启动后再启动
    depends_on:
      - zk
    ports:
      - 8080:8080
    environment:
      - admin.registry.address=zookeeper://zk:2181
      - admin.config-center=zookeeper://zk:2181
      - admin.metadata-report.address=zookeeper://zk:2181

# 4. 其他

推荐一个非官方的: Docker Hub

version: '3'
services:
  zk:
    image: zookeeper
    container_name: zk
    ports:
      - 2181:2181
  dubbo-admin:
    image: chenchuxin/dubbo-admin
    container_name: dubbo-admin
    # 等待zk启动后再启动
    depends_on:
      - zk
    ports:
      - 8080:8080
    environment:
      - dubbo.registry.address=zookeeper://zk:2181
      - dubbo.admin.root.password=root
      - dubbo.admin.guest.password=guest
