# Docker下RabbitMQ的使用

Docker下RabbitMQ的使用及集群搭建

# 2. 启动

查看官网: https://www.rabbitmq.com/download.html，直接有启动的命令

执行命令 docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3-management 启动

启动成功，然后访问: http://localhost:15672，用guest/guest登录就可以打开管理的 Web 界面
